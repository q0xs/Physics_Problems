Dual Friction Sources

The lower block (\(m_2\)) is sliding against two surfaces simultaneously. Since the upper block (\(m_1\)) is tethered to the wall, it cannot move, creating relative motion at the interface.

Interface Friction: \(f_{12} = \mu_k (m_1 g)\)
Floor Friction: \(f_{f} = \mu_k (m_1 + m_2)g\)

Equation of Motion

\[F - f_{12} - f_f = m_2 a\]

Note: If \(F < f_{12} + f_f\), the acceleration is 0 as static friction would hold the block in place.
