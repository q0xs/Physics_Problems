Theory

The damped oscillator is governed by $m\ddot x + b\dot x + kx = 0$.

The critical damping value is $b_c = 2\sqrt{mk}$.

For $b < b_c$ the motion is underdamped, for $b = b_c$ critically damped, and for $b > b_c$ overdamped.

The numerical solver uses the first-order system $\dot x = v$ and $\dot v = -\frac{b}{m}v - \frac{k}{m}x$ integrated with RK4.
