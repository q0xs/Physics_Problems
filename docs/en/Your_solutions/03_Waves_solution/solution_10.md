# Task 10 – Animation of Multiple Wave Sources

## Problem Statement

Prepare an HTML animation in which dots can be placed on the screen and used as sources of waves described by

$$
u(\vec{r},t) = \frac{A}{|\vec{r}-\vec{r_0}|^\alpha} \sin(k |\vec{r} - \vec{r_0}| - \omega t)
$$

where $\vec{r_0}$ is the source position and $\alpha \in [0,2]$. The animation should display the superposition of waves from all sources.

## Theory

For a source at position $\vec{r_0}$, the distance from the source to an observation point $\vec{r}$ is

$$
R = |\vec{r} - \vec{r_0}|
$$

The partial wave generated by one source is

$$
u_i(\vec{r},t) = \frac{A}{R_i^\alpha} \sin(kR_i - \omega t)
$$

For multiple sources, the principle of superposition gives

$$
u(\vec{r},t) = \sum_i u_i(\vec{r},t)
$$

The parameter $\alpha$ controls amplitude decay:

- $\alpha = 0$ gives no radial decay,
- larger $\alpha$ increases attenuation with distance.

The wave number is related to the wavelength by

$$
k = \frac{2\pi}{\lambda}
$$

and the angular frequency is

$$
\omega = 2\pi f
$$

## Step-by-Step Solution

### Model of the Simulation

Each mouse click adds a point source at a chosen location. For each pixel of the display grid, the code computes the sum

$$
u(\vec{r},t) = \sum_i \frac{A}{R_i^\alpha} \sin(kR_i - \omega t)
$$

A small cutoff is used near the source position to avoid numerical divergence when $R_i \to 0$.

### Visualization

The resultant displacement is mapped to a color scale. Positive and negative values are displayed using different intensities, which reveals constructive and destructive interference between sources.

### Effect of the Parameter $\alpha$

The attenuation exponent changes the spatial structure of the field.

- Small $\alpha$ produces strong long-range interference patterns.
- Large $\alpha$ localizes each source more strongly and reduces distant contributions.

## Final Result

The following HTML file implements:

- interactive placement of sources by mouse clicks,
- a slider for $\alpha$ in the interval $[0,2]$,
- real-time superposition of all waves.

## Interpretation

This model illustrates the superposition principle for circular wavefronts emitted by multiple sources. It also demonstrates how geometric attenuation modifies the observed field while preserving interference effects.
