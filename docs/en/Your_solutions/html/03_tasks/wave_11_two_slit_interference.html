Waves — Section 3

Task 11 — Young's Two-Slit Interference

Observe real-time interference from two coherent point sources while changing slit spacing and wavelength.

Interference Controls

Constructive condition
$\Delta R = m\lambda$
Destructive condition
$\Delta R = \left(m+\frac12\right)\lambda$

Real-Time Interference Pattern

Physics of the Pattern

The two coherent sources produce partial waves

$u(\vec r,t)=\frac{A}{R_1}\sin(kR_1-\omega t)+\frac{A}{R_2}\sin(kR_2-\omega t)$.

Changing the slit spacing modifies the path-difference gradient across the screen. Changing the wavelength changes fringe spacing and temporal behavior.
