Task 07 — Standing Wave Modes

Relate the number of antinodes on a string to the wavelength of the standing mode.

Interactive Controls

Computed Results

Length in meters
0.8000 m
Wavelength
0.4000 m
Mode relation
$L = n\frac{\lambda}{2}$ and the number of antinodes equals $n$ for a string fixed at both ends.

Step-by-Step Physics

Mode condition

Each antinode corresponds to one half-wavelength segment along the string.

Formula

$L=n\frac{\lambda}{2}$, so $\lambda = \frac{2L}{n}$.

Interpretation

Higher modes fit more half-wavelengths into the same physical length, so the wavelength decreases as the mode number increases.
