# Task 07 – Standing Wave Modes

## Problem Statement

A standing wave with four antinodes is produced on a string of length $L = 80 \, \text{cm}$. Determine the wavelength of this mode.

## Theory

For a string fixed at both ends, the standing-wave condition is

$$
L = n \frac{\lambda}{2}
$$

where $n$ is the harmonic number. For a string with fixed ends, the number of antinodes is equal to $n$.

## Step-by-Step Solution

Given:

- $L = 80 \, \text{cm} = 0.80 \, \text{m}$
- number of antinodes $= 4$

Therefore,

$$
n = 4
$$

Use the standing-wave condition:

$$
L = n \frac{\lambda}{2}
$$

Substitute the known values:

$$
0.80 = 4 \frac{\lambda}{2}
$$

$$
0.80 = 2 \lambda
$$

$$
\lambda = \frac{0.80}{2}
$$

$$
\lambda = 0.40 \, \text{m}
$$

## Final Result

$$
\lambda = 0.40 \, \text{m}
$$

## Interpretation

A mode with four antinodes corresponds to the fourth harmonic. The string contains four half-wavelength segments along its length, which reduces the wavelength compared with the fundamental mode.
