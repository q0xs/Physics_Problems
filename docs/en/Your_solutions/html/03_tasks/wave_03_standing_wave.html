Task 03 — Superposition Principle

Visualize how two opposite traveling waves combine into a standing wave and identify node positions.

Standing-Wave Controls

Standing wave
$y(x,t) = 2A \sin(kx) \cos(\omega t)$
Node positions
$x_n = n\frac{\lambda}{2}$

Animated Standing Wave

Nodes are fixed
They remain at zero displacement for all time.
Antinodes oscillate maximally
They are halfway between adjacent nodes.
Origin of standing wave
Two equal waves moving in opposite directions superpose.

Derivation

Start with $y_1(x,t) = A\sin(kx-\omega t)$ and $y_2(x,t) = A\sin(kx+\omega t)$.

Using $\sin u + \sin v = 2\sin\left(\frac{u+v}{2}\right)\cos\left(\frac{u-v}{2}\right)$ gives

$y(x,t) = 2A\sin(kx)\cos(\omega t)$.

The nodes satisfy $\sin(kx)=0$, hence $kx=n\pi$ and $x=n\frac{\lambda}{2}$.
