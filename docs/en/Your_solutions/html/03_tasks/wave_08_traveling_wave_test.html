Waves — Section 3

Task 08 — Which Functions Can Describe a Traveling Wave?

Classify candidate functions by checking their structure against the wave equation.

Wave-Equation Test

A function describes a traveling wave when it can be written as $f(x-vt)$ or $f(x+vt)$ and satisfies

$\frac{\partial^2 y}{\partial x^2} = \frac{1}{v^2}\frac{\partial^2 y}{\partial t^2}$

a) $A\cos(kx^2-\omega t)$
Fails because the argument contains $x^2$, not $x\pm vt$.
b) $A(x-vt)^2$
Valid because it is exactly of the form $f(x-vt)$.
c) $A\log(x+vt)$
Valid because it is exactly of the form $f(x+vt)$.

Reasoning

Any sufficiently smooth function of $x-vt$ or $x+vt$ satisfies the one-dimensional wave equation.

For $y(x,t)=f(\xi)$ with $\xi = x-vt$, differentiation gives

$\frac{\partial y}{\partial x}=f'(\xi)$ and $\frac{\partial y}{\partial t}=-vf'(\xi)$.

Differentiating again yields

$\frac{\partial^2 y}{\partial x^2}=f''(\xi)$ and $\frac{\partial^2 y}{\partial t^2}=v^2 f''(\xi)$.

Hence $\frac{\partial^2 y}{\partial x^2}=\frac{1}{v^2}\frac{\partial^2 y}{\partial t^2}$.

Classification Summary

a)
Not a traveling wave
b)
Traveling wave
c)
Traveling wave
