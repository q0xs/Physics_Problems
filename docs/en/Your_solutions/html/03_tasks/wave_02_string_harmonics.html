Step-by-Step Physics

Fundamental mode

For a string fixed at both ends, the lowest mode contains half of a wavelength on the string.

Therefore $L = \frac{\lambda_1}{2}$, so $\lambda_1 = 2L$.

Wave speed

Once the wavelength is known, the speed follows from $v = f\lambda$.

Interpretation

The wave speed depends on string tension and linear mass density. The frequency changes between harmonics, but the string itself sets the propagation speed.
